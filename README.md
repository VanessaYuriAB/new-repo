## 📌 Sobre o projeto

Este projeto foi criado como parte dos meus primeiros estudos em desenvolvimento front-end
e marcou **meu primeiro contato prático com Git e GitHub**.

O objetivo principal foi consolidar conceitos fundamentais de:

- Estruturação de páginas com HTML semântico
- Estilização básica com CSS
- Versionamento de código utilizando Git
- Criação de commits incrementais e descritivos
- Fluxo básico de desenvolvimento com repositório remoto (GitHub)

Este repositório representa o início da minha trajetória na área e reflete boas práticas
aprendidas desde os primeiros passos.

---

## 🧠 O que aprendi

Durante o desenvolvimento deste projeto, pratiquei e aprendi:

- Estruturação de páginas usando **HTML5 semântico**
  - Uso de tags como `<header>`, `<main>`, `<section>`, `<aside>`, `<footer>` e `<time>`
- Estilização básica com **CSS3**
- Criação e organização de arquivos em um projeto front-end
- Inicialização e configuração de um repositório Git
- Uso dos principais comandos do Git:
  - `git init`
  - `git status`
  - `git add`
  - `git commit`
  - `git push`
- Publicação de um projeto local em um repositório remoto no **GitHub**

## 🚀 Como visualizar o projeto

1. Clone este repositório:

```Shell
git clone git@github.com:VanessaYuriAB/new-repo.git
```

2. Acesse a pasta do projeto:

```Shell
cd sample-project
```

3. Abra o arquivo `index.html` no navegador. Você também pode usar a extensão
   `Live Server` do `VS Code`.
