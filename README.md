# My Sample Project for Commits - HTML, CSS & Git

> Projeto educacional desenvolvido no início da minha jornada como desenvolvedora front-end.

## 📌 Sobre o projeto

Projeto inicial de aprendizado desenvolvido para consolidar fundamentos de HTML, CSS e Git.

Este foi o meu primeiro projeto publicado no GitHub. 

O objetivo foi praticar:
- Estruturação de páginas com HTML semântico
- Estilização básica com CSS
- Versionamento de código com Git
- Criação de commits incrementais

## 🛠 Tecnologias utilizadas
- HTML5
- CSS3
- Git

## 📚 O que aprendi
- Uso de tags semânticas como `<aside>`, `<footer>` e `<time>`
- Criação e organização de commits
- Publicação e versionamento de um projeto no GitHub

## ✅Status do projeto
Projeto concluído (educacional).

## 📝 Observação
Este projeto faz parte do meu processo de aprendizado e marca o início da minha trajetória no desenvolvimento web.
